# Captures d'écran / Screenshots

Les fichiers de ce dossier sont référencés par les README (`README.md` et `README.en.md`).

Déposez ici les deux captures avec **exactement** ces noms :

| Fichier | Thème | Affiché pour |
| :--- | :--- | :--- |
| `screenshot-dark.png` | Sombre | Lecteurs en mode sombre (`prefers-color-scheme: dark`) |
| `screenshot-light.png` | Clair | Lecteurs en mode clair + repli par défaut |

GitHub choisit automatiquement la bonne image selon le thème du lecteur grâce à la balise `<picture>` des README.

> Largeur recommandée : ~1450 px (les captures sont affichées à 900 px de large).
